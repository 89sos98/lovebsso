
Web ： 主项目

文件夹：
App_Data: 存放access 数据库
Editor  : 编辑器的文件
Setting : html快速解析成xml的dtd文档
Upload  : 用户上传的一些图片等文件

处理程序：

Admin.ashx   : 处理用户后台程序


Index.ashx   : 处理用户首页程序
Article.ashx : 处理用户文章程序
Photo.ashx   : 处理用户图片程序

Home.ashx    : 处理系统主站程序
RegLogin.ashx: 处理系统主站的用户注册/登陆/退出的处理程序
Language.ashx: 处理系统主站的多语言的切换的处理程序
Error.ashx   : 处理系统出错时的处理程序

Web.config   : 系统配置文件
