CYQ.Editor 是一套简单的编辑器，使用于后台的文章编辑中。

基础说明：
1：Editor.cs 基础类，被调用生成编辑器的html代码
2：ToolBar.cs 工具栏类，代码有点多，比较烦。
3：Uploader.cs 编辑器中上传图片和flash的。

对应的css和html文件在 web项目中的Editor目录下。

使用示例：见Logic中的FillAdminArticle类。

			//填充编辑器
            Editor editor = new Editor(Document, Language);
            editor.EditorID = "txtBody";
            editor.Width =700;
            editor.Height = 400;
            editor.BasePath = Config.HttpHost;
            if (withEdit)
            {
                FillEdit(editor);
            }
